# Installation Instructions

## To install react-icons:

### Option 1: Run the batch file
Double-click `install-react-icons.bat` in the project root folder.

### Option 2: Run in terminal
```bash
npm install react-icons
```

### Option 3: If npm doesn't work due to PowerShell restrictions
1. Open Command Prompt (cmd.exe) as Administrator
2. Navigate to the project folder:
   ```
   cd "c:\Users\11\Desktop\Projects\hair salon"
   ```
3. Run:
   ```
   npm install react-icons
   ```

## After installation:
The sidebar icons will automatically update to use React Icons (Feather Icons) which blend better with the website theme.

### Icons Used:
- **Dashboard**: FiHome (home icon)
- **Inventory**: FiPackage (package box icon)
- **Orders**: FiShoppingCart (shopping cart icon)
- **Logout**: FiLogOut (logout arrow icon)

These icons are from the Feather Icons collection, which provides clean, minimalist icons that match professional web applications.
